# 프로젝트 기획의도

내가 좋아하는 영화 예매와 해당 굿즈들을 한 번에 보고 싶은데?
하나로 묶여있는 페이지가 없을까? 라는 질문들에서 시작된 프로젝트
 
이러한 생각들을 모두 해소 시킬 수 있도록,  일반 고객들부터 영화를 주로보는 매니아 개층까지 영화를 예약하는 것 뿐만이 아닌 다양한 컨텐츠를 추가시킨 사이트를 만들기로함
